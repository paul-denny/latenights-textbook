# Linear Search
Linear search is perhaps the simplest way of searching for data in any kind of linear data type.
Linear search can be used to find the index of certain items in a sequence or to check whether a particular item exists within an iterable.
The premise of this algorithm is simple, loop over values and check each one until you find the value you're looking for:

## Benefits of Linear Search
Linear search is a very simple algorithm and is quick and easy to implement.
While it is not always the fastest searching algorithm, it can search nominal (i.e. unsortable) or unsorted data without preprocessing which many sorting algorithms cannot do.
